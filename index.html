Лабораторна робота №1

Тема, мета, посилання

Тема: Створення статичного веб-сайту для сервісу оренди автомобілів (Das Auto Ukraine). Семантична розмітка, навігація, секції: hero, about, services, team, contacts.

Мета: Реалізувати адаптивну HTML-сторінку з семантикою та структурою для презентації бізнесу — оренди авто; підключити зовнішній CSS для стилізації.

Посилання

  • Репозиторій WEB-застосунку: https://github.com/sobchukvm-zpi51-coder/Sobchuk-Vladyslava-Project-2
  • Головні файли проекту: index.html, about.html, services.html, contacts.html, team.html, css/style.css

Структура сайту (коротко)

Сайт побудований семантично: <header> — навігація та логотип; <main> — секції hero, about, services, team, contacts; <footer> — підвал. Навігація використовує внутрішні посилання на html документи.

Головна сторінка — короткий огляд

Головна сторінка містить: hero-блок із CTA, короткий блок про компанію, блоки послуг (Оренда авто, Оренда з водієм, Довгострокова оренда), секцію команди та контакти. Усі стилі винесені в css/style.css. Зображення розміщені в папці images/.

Тема ЛР №2

Стильове оформлення елементів в HTML-документах. Підключення CSS. Селектори (тег, клас, id) та основні правила стилізації.

Мета ЛР №2

Навчитися підключати зовнішні CSS-файли та застосовувати селектори для оформлення елементів інтерфейсу сайту оренди авто.

Посилання

  • Репозиторій: sobchukvm-zpi51-coder/Sobchuk-Vladyslava-Project-2

Способи підключення стилів

1) Зовнішній CSS-файл (використано в проєкті)

<link rel="stylesheet" href="css/style.css">

2) Внутрішній та інлайн (не використані як основні)

<style> /* внутрішні стилі */ </style>
<h1 style="color:#00595C;">Заголовок</h1> /* інлайн */

Фрагменти з css/style.css (взято з вашого проєкту)

/* базові налаштування */
body {
    text-align: center;
    margin: 0;
    font-family: Arial, sans-serif;
}

/* header */
header {
    display: flex;
    justify-content: space-between;
    align-items: left;
    padding: 15px 30px;
}

/* лого */
.logo img {
    width: 80px;
}

/* описи послуг */
.service-card {
    border: 1px solid #ddd;
    padding: 20px;
    margin: 15px auto;
    max-width: 400px;
    border-radius: 10px;
}

Ці стилі показують підхід: базові правила для body, стилізація header, обмеження ширини та стилі для карток послуг.

Висновки до ЛР №2

У ході роботи застосовано підключення зовнішнього CSS, продемонстровано приклади селекторів та стилів з реального проєкту (header, .service-card). Це дозволяє оформити сайт оренди авто у відповідній візуальній гамі та підтримувати централізовану стилізацію.
